# 教师端 (Teacher End) 全局页面设计规范

严格遵循无 Emoji、全令牌化、“青电×琥珀”双色规范。
所有页面同时遵循本目录 `README.md` 的公共契约。
**更新说明**：
1. 顶栏已取消全局搜索，左侧为 Logo，右侧固定为「下载任务中心」、「消息通知中心」与带下拉详情的「个人头像」。
2. 通知、下载、个人中心功能均由顶栏直接触发进入，不再占用侧边栏位置。
3. 涵盖《前端后端功能对齐清单》中所有教师端需要的 32 个主入口与深层页面。

**实现基准**：
1. 本文是 UI/UX 视觉与页面结构草图，最终实现以后端已实现接口、DTO、权限和 `docs/前端后端功能对齐清单.md` 为准。
2. 页面示例数据、进度、状态和文案只表达布局意图；实现时必须接 `@chaimir/api-client`，不得用模拟数据替代真实后端能力。
3. 教师端仿真只做提交、更新、预览和维护；审核通过、驳回、归档、重发只属于平台端。
4. 最终界面不得使用 emoji，状态必须使用 Lucide 图标、`@chaimir/ui` 状态组件和文字共同表达。
