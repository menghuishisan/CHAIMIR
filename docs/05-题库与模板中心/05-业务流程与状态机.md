# M5 题库与模板中心 — 业务流程与状态机

> Mermaid 描述内容创作发布、版本演进、克隆共享、组卷、引用取用。
> 最后更新:2026-05-29

---

## 1. 内容创作与发布

```mermaid
flowchart TD
    A[教师创建内容] --> B[选类型: 实验模板/竞赛题/理论题]
    B --> C[填元信息 + 类型化内容体]
    C --> D[草稿态,自由编辑]
    D --> E{发布?}
    E -->|是| F[定版 version, status=已发布]
    F --> G[可被业务引用]
    E -->|否| D
```

---

## 2. 内容状态机

```mermaid
stateDiagram-v2
    [*] --> 草稿: 创建
    草稿 --> 已发布: 发布(定版)
    已发布 --> 已弃用: 弃用(不可被新引用)
    草稿 --> 已删除: 软删(无引用)
    已弃用 --> [*]
    已删除 --> [*]
```

- 已发布版本内容**不可变**;改内容走"发新版本"。
- 有引用的内容不可删,只能弃用(旧引用仍指向该版本)。

---

## 3. 版本演进

```mermaid
flowchart LR
    A[内容 code v1.0 已发布] --> B[业务X引用锁定 v1.0]
    A --> C[教师改内容]
    C --> D[发布 v1.1]
    D --> E[新业务Y引用 v1.1]
    B --> F[业务X 仍用 v1.0 不受影响]
```

> 引用锁版本保证旧实验/旧竞赛可复现稳定。

## 4. 克隆流程

```mermaid
flowchart TD
    A[选择克隆源: 自己的/共享库的] --> B[复制外壳 + 内容体]
    B --> C[生成新 code, status=草稿]
    C --> D[清空: 引用关系/学生数据/提交/复用统计]
    D --> E[成为克隆者独立内容, 与源无关联]
```

---

## 5. 共享库流程

```mermaid
flowchart TD
    A[教师内容 已发布] --> B[设为共享 visibility=shared]
    B --> C[进入跨校共享库, 全平台可见]
    C --> D[他校教师浏览 → 克隆到本校]
    D --> E[克隆副本独立, 原作者取消共享不影响副本]
```

## 6. 组卷流程

```mermaid
flowchart TD
    A[创建卷] --> B{组卷方式}
    B -->|手动| C[逐题选择 + 设分值]
    B -->|按条件随机| D[设条件: 知识点/难度/数量]
    D --> E[系统从题库随机抽题]
    C --> F[生成卷: 题目引用集合 + 分值, 锁定版本]
    E --> F
    F --> G[被 M6 作业 / M8 竞赛引用]
```
